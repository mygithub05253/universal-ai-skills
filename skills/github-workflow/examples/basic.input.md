# 예시 입력 — 기본 (새 기능, 전체 흐름)

회의실 예약 웹앱에 **예약 생성 API**를 추가하려 한다. Claude Code에서 실행.

- `{{CHANGE_DESCRIPTION}}`: 회의실 예약 생성 API 추가 (POST /reservations)
- `{{BASE_BRANCH}}`: (비움 → main)
- `{{CHANGE_TYPE}}`: feat
- `{{ISSUE_REF}}`: #12
- `{{MERGE_METHOD}}`: (비움 → squash)
- `{{TEAM_RULES}}`: (비움 → 머지 게이트에서 질문)

현재 상태: `main`에서 작업 중이며 워킹 트리는 깨끗하다. 코드 작업은 이 흐름 안에서 진행한다.
