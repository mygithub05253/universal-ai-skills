## 출력 규칙
- 순수 Markdown 으로만 출력합니다. 코드펜스로 전체를 감싸지 않습니다.
- 기능을 과장하지 않고 입력된 사실만 기술합니다. 모르는 정보는 추측 대신 `<!-- TODO -->` 로 남깁니다.
- 실행 명령은 입력된 기술 스택에 맞게 구체적으로 작성합니다 (예: Next.js 면 `npm run dev`).
- 배포 URL 이 비어 있으면 데모 링크 섹션을 생략합니다.
